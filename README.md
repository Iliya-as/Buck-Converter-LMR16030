# DC-DC Buck Converter (LMR16030)

## Overview

A compact, non-isolated DC-DC Buck (Step-Down) converter based on the **LMR16030** switching regulator IC. The board converts a **5–50 VDC** input into an adjustable output voltage while delivering up to **3 A** of output current. The design is **optimized for a typical input voltage of 30 VDC**. Input protection is provided by a **TVS diode** and an **input fuse**.

> *The maximum output voltage depends on the input voltage and operating conditions.

---

## Features

- Adjustable output voltage
- Up to **3 A** output current
- Approximately **500 kHz** switching frequency
- Compact **50 × 43 mm** PCB
- Optimized for **30 VDC** nominal input
- TVS diode for input transient protection
- Input fuse for overcurrent protection
